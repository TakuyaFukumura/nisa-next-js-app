# スマホ対応設計資料

## 概要

スマートフォン画面でのユーザー体験を改善するための設計資料です。  
以下の3点を対象に仕様を定義します。

1. ハンバーガーメニューの導入
2. スマホ表示時の外側余白削減
3. グラフ付随文字の見切れ解消

---

## 1. ハンバーガーメニューの導入

### 現状

`src/app/components/Header.tsx` において、ナビゲーションリンク（全体・年別）はすべての画面幅で常時表示されています。  
スマホ画面では横幅が狭いため、タイトル・ナビ・テーマ切替ボタンが横一列に並んで窮屈な表示になります。

### 方針

- **ブレークポイント**: Tailwind の `md`（768px）を基準とし、それ未満をスマホ扱いとします。
- **ナビゲーション表示切替**:
  - `md` 未満: 通常のナビリンクを非表示にし、ヘッダー左側にハンバーガーアイコンを表示します。
  - `md` 以上: 現状通りナビリンクを表示し、ハンバーガーアイコンを非表示にします。

### ハンバーガーアイコンの仕様

- 位置: ヘッダー左端（タイトル「nisa」の左隣）
- 表示形式: 三本線アイコン（`☰`）
- クリック動作: ドロワーメニューの開閉トグル

### ドロワーメニューの仕様

- 表示形式: ヘッダー直下にスライドイン（縦方向ドロップダウン）
- 表示内容: 現在のナビリンクと同等（全体・年別）
- アクティブ状態の表現: 既存の `navLinkClass` ロジックをそのまま流用します。
- 閉じる条件:
  - ナビリンクをクリックしたとき
  - メニュー外をクリックしたとき
  - ハンバーガーアイコンを再クリックしたとき

### 実装イメージ（擬似コード）

```tsx
// Header.tsx への追加イメージ
const [menuOpen, setMenuOpen] = useState(false);

// ハンバーガーアイコン（md未満で表示）
<button
  className="md:hidden flex items-center p-2"
  onClick={() => setMenuOpen(!menuOpen)}
  aria-label="メニューを開く"
>
  ☰
</button>

// 既存のナビリンク（md未満で非表示）
<nav className="hidden md:flex items-center gap-2">
  ...
</nav>

// ドロワーメニュー（md未満かつmenuOpenがtrueの場合に表示）
{menuOpen && (
  <div className="md:hidden absolute top-16 left-0 w-full bg-white dark:bg-gray-800 ...">
    <nav className="flex flex-col p-4 gap-2">
      ...
    </nav>
  </div>
)}
```

### 改修対象ファイル

| ファイル | 変更内容 |
|---|---|
| `src/app/components/Header.tsx` | ハンバーガーアイコン追加、ドロワーメニュー追加、既存ナビの表示切替 |

## 2. スマホ表示時の外側余白削減

### 現状

各ページのルートコンテナに `p-6`（padding: 1.5rem / 24px）が固定指定されています。  
スマホ画面ではこの余白がグラフ表示領域を圧迫しています。

対象ファイルと該当箇所：

| ファイル | 現在のクラス |
|---|---|
| `src/app/page.tsx` | `p-6` |
| `src/app/yearly/page.tsx` | `p-6` |
| `src/app/yearly/[year]/page.tsx` | `p-6` |

### 方針

Tailwind のレスポンシブプレフィックスを利用し、スマホ時の余白を縮小します。

| 画面幅 | 現在 | 変更後 |
|---|---|---|
| デフォルト（スマホ） | `p-6` (24px) | `p-2` (8px) |
| `sm` 以上（640px〜） | `p-6` (24px) | `sm:p-4` (16px) |
| `lg` 以上（1024px〜） | `p-6` (24px) | `lg:p-6` (24px) |

### 変更イメージ

```tsx
// 変更前
<div className="... p-6">

// 変更後
<div className="... p-2 sm:p-4 lg:p-6">
```

### 内側カードの余白も調整

カード内の `p-6` も同様に見直します。

```tsx
// 変更前
<div className="bg-white dark:bg-gray-800 rounded-2xl shadow-lg p-6 mb-6">

// 変更後
<div className="bg-white dark:bg-gray-800 rounded-2xl shadow-lg p-3 sm:p-6 mb-4 sm:mb-6">
```

### 改修対象ファイル

| ファイル | 変更内容 |
|---|---|
| `src/app/page.tsx` | ルートコンテナおよびカードの余白をレスポンシブ対応に変更 |
| `src/app/yearly/page.tsx` | 同上 |
| `src/app/yearly/[year]/page.tsx` | 同上 |

## 3. グラフ付随文字の見切れ解消

### 現状

グラフコンポーネントで `label` プロパティを用いた外周ラベルを表示していますが、  
スマホ画面では表示エリアが狭くラベルがコンテナ外に見切れることがあります。

#### 問題箇所

**`NisaOverallChart.tsx`（円グラフ）**
```tsx
label={({name, value}) => `${name}: ${formatAmount(value)}`}
```
- ドーナツグラフ外周にテキストラベルを描画しているため、幅が狭いとはみ出します。

**`NisaYearlyDetailChart.tsx`（円グラフ）**
```tsx
label={({name, value}) => `${name}: ${formatAmount(value)}`}
```
- 同上。`grid grid-cols-1 sm:grid-cols-2` のためスマホでは1列になりますが、横幅が狭い場合にラベルが見切れます。

**`NisaYearlyChart.tsx`（棒グラフ）**
```tsx
label={{
    value: `年間上限 ${formatAmount(YEARLY_TOTAL_LIMIT)}`,
    position: 'insideTopRight',
    ...
}}
```
- `ReferenceLine` のラベルが棒グラフ右端に重なり、スマホでは表示が狭くなります。

### 方針

#### 円グラフ（NisaOverallChart・NisaYearlyDetailChart）

外周ラベルを廃止し、`Legend`（凡例）と下部テキストエリアのみで金額・利用率を表現します。  
これにより、グラフ表示エリア内に情報が収まります。

```tsx
// 変更前
<Pie
    ...
    label={({name, value}) => `${name}: ${formatAmount(value)}`}
>

// 変更後
<Pie
    ...
    label={false}
>
```

金額情報はすでにグラフ下部のテキストエリアに表示されているため、  
`Legend` + 下部テキストで情報が欠落しない設計とします。

#### 棒グラフ（NisaYearlyChart）

`ReferenceLine` のインラインラベルをテキストのみの表示からより小さいフォントに変更するか、  
ラベルの `position` を `'insideTopLeft'` に変更して、狭い画面でも収まるようにします。

```tsx
// 変更案
label={{
    value: `上限 ${formatAmount(YEARLY_TOTAL_LIMIT)}`,
    position: 'insideTopLeft',
    fill: '#ef4444',
    fontSize: 11
}}
```

また、棒グラフの `margin` の `right` 値を増やすことで、ラベルの右端が切れにくくなります。

```tsx
// 変更前
<BarChart data={chartData} margin={{top: 10, right: 30, left: 20, bottom: 5}}>

// 変更後
<BarChart data={chartData} margin={{top: 10, right: 10, left: 10, bottom: 5}}>
```

### 改修対象ファイル

| ファイル | 変更内容 |
|---|---|
| `src/app/components/NisaOverallChart.tsx` | 外周ラベルを非表示に変更 |
| `src/app/components/NisaYearlyDetailChart.tsx` | 外周ラベルを非表示に変更 |
| `src/app/components/NisaYearlyChart.tsx` | ReferenceLine ラベルの位置・フォントサイズ調整 |
